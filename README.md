## 🚀 Overview
This project is a single-file, self-contained HTML presentation that covers the history, importance, structure, and legal framework of the Panchayati Raj system. It features a modern **Glassmorphism UI**, smooth animations, and a responsive layout designed for full-screen classroom presentations.

## ✨ Key Features
- **Modern UI**: Dark-themed glassmorphism design with a dynamic Ashoka Chakra background.
- **Interactive Navigation**: Seamlessly navigate through 10 slides using Arrow keys or Navigation dots.
- **3-Tier System Visualization**: Horizontal flow diagram showing the District, Block, and Village levels.
- **Iconography**: Clean, semantic SVG icons replacing standard emojis for a professional look.
- **Responsive Design**: Optimized for full-screen (F11) mode on any presentation display.
- **Zero Dependencies**: Entirely self-contained in a single `.html` file (uses Google Fonts).

## 📊 Presentation Structure
1. **Author Slide**: Personal profile and social links.
2. **Hero/Welcome**: Title and central theme.
3. **What is it?**: Introduction to local self-governance.
4. **Importance**: Why decentralization matters.
5. **The 3-Tier System**: Structural overview.
6. **Gram Panchayat**: Deep dive into village-level functions.
7. **Samiti & Parishad**: Block and District level details.
8. **73rd Amendment**: The constitutional turning point (1992).
9. **Conclusion**: The soul of Indian democracy.
10. **Thank You**: Final closing slide.
